# Ville Partagée

## Une plateforme de collaboration citoyenne qui partage, liste et permet de s'inscrire à des annonces d'événements ou prêts.

### Liste des étudiants

- Étudiant 1 : TAURAND Clément
    - US1 : Création d'un user
    - US2 : Création d'une annonce
    - US3 : Données sur une période donnée
    - US4 : Données en temps réel
    - US5 : Page de modération
    - US6 : Historique des modérations par utilisateur
    - US7 : Algorithme de modération
    - US8 : Page des annonces créées
    - US9 : Ouverture d'une annonce depuis la carte
- Étudiant 2 : MEUNIER Gilles
    - US1 : Profil de l'utilisateur avec ses données
    - US2 : Profil de l'annonce avec ses données
    - US3 : Profil de l'utilisateur avec ses consultations
    - US4 : Mock les consultations
    - US5 : Calcul des annonces les plus pertinentes
    - US6 : Sélection et affichage des annonces
- Étudiant 3 : TRAN Louis
    - US1 : Afficher la carte
    - US2 : Afficher les quartiers
    - US3 : Affichage des lieux (emplacements possibles des annonces)
    - US4 : Différents affichages selon le nombre d'annonces visibles
    - US5 : Recentrage de la carte sur le lieu sélectionné
    - US6 : Afficher toutes les annonces
    - US7 : Affichage avec filtre (tag, quartier) des annonces
    - US8 : Déplacement automatique en fonction du filtre par quartier
    - US9 : Monter seulement les annonces filtrés
    - US10 : Afficher le nombre d'annonces
